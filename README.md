## Run Locally

Clone the project

```bash
  git clone https://github.com/sahadsalim/Nest-JS-Bidding-API-Service.git
```

Go to the project directory

```bash
  cd bidding-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
